# car-price-prediction
Linear model running to predict car prices

Dataset Description:

The dataset has 9 Features and 301 datapoints.
The columns are :

Car_Name        Name of the car,\n
Year            Year of manufacturing,\n
Selling_Price	  Initial price of sale,\n
Present_Price	  Current price of sale,\n
Kms_Driven      Total distance travelled,\n
Fuel_Type       Type of fuel used,\n
Seller_Type     Professional seller or individual seller,\n
Transmission	  Type of transmission,\n
Owner           Current owner,\n

Our aim is to prepare a model that predicts the current selling price of the cars.
